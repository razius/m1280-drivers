void spi_init ( uint8_t  mode,
uint8_t  data_order,
uint8_t  clock_polarity,
uint8_t  clock_phase,

)

Initializes the SPI driver and sets the mode for it.

Note
Must be called before any other method can be used.
Parameters
modeSets the operation mode of the SPI interface.
Valid choices are SPI_MODE_MASTER and SPI_MODE_SLAVE.
data_orderSelects which bit will be transmitted first.
Valid choices are SPI_DATA_ORDER_LSB_FIRST and SPI_DATA_ORDER_MSB_FIRST.
clock_polaritySelects the output when SCK is idle.
Valid choices are SPI_CLOCK_POLARITY_HIGH_WHEN_IDLE and SPI_CLOCK_POLARITY_LOW_WHEN_IDLE.
clock_phaseSelects when the data should be sampled.
Valid choices are SPI_CLOCK_PHASE_SAMPLE_LEADING and SPI_CLOCK_PHASE_SAMPLE_TRAILING.
*callback_function()Callback function to be executed when the transfer is complete.
The Global Interrupt Enable bit in SREG must be set.
